# Others

Ce petit script tout simple est un gestionnaire de contact en python.
# Fonctionnalité : 
Ce script permet de : 
- Ajouter des contacts 
- Suprrimer des contacts 
- Modifier des contacts 
- Afficher la liste des contacts
- Recherche par numéro ou par nom si le contact existe dans la base
